## 環境
Gulp
※node_modulesは外しています

## Gulpの仕様
-sassの記述はsrcフォルダの中で行う

-画像はsrcフォルダのimagesの中に格納する（圧縮される）

-コンパイルされたCSSと圧縮された画像は第一階層のCSSフォルダ / imagesの中に出力される
